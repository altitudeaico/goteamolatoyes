How to Host Your App

Put your creation on the internet so anyone can visit it!

← Back to Code Quest

🤔 Wait — what does "hosting" mean?

When Claude makes you a webpage, the code lives on YOUR screen — nobody else can see it yet. Hosting means putting that code somewhere on the internet so it gets its own link (like yourname.github.io) that the whole family can visit! Think of it like this:

💡

The Big Idea

🖊️
Your Claude code Like drawing a picture in your notebook — only you can see it
🖼️
Hosted app Like hanging that picture in a gallery — everyone can come and see it!
🏠
GitHub Pages The free "gallery wall" where your app lives — and we already use it!
🔗
Your link The address people type to visit — like yourname.github.io/myapp
🤖

Step 1 — Ask Claude to make your page

Do this in Claude.ai

First, use Claude to build your webpage. Copy this prompt and fill in YOUR details:

Make me a fun personal webpage about me! My name is: [YOUR NAME] My favourite colour is: [A COLOUR] I love: [YOUR HOBBY OR INTEREST] My team is: [GIRLS or BOYS] Make it colourful with big text, emojis, and animations. At the bottom, add "Built with AI at Code Quest 🚀" Show me the full HTML code.
💡 Tip: The more detail you give Claude, the cooler your page will be! Tell it your favourite TV show, food, colour — anything!

Claude will give you a big block of code that starts with <!DOCTYPE html>. That's your webpage — now you just need to put it online.

🐙

Step 2 — Put it on GitHub Pages

Do this with a parent's help

GitHub Pages is a free service that hosts websites. Uncle B has already set up the family's GitHub account — you'll add your page to it.

  • 1
    Go to the family GitHub Open github.com/altitudeaico/goteamolatoyes — ask a parent to log in if needed.
  • 2
    Go into the "hackathon" folder Click the folder called hackathon — this is where all the Code Quest files live.
  • 3
    Create a new file Click the "Add file" button, then choose "Create new file".
  • 4
    Name your file In the filename box, type your name followed by .html — for example: elsie.html or theodore.html
  • 5
    Paste your Claude code Copy ALL the code Claude gave you (from <!DOCTYPE html> to </html>) and paste it into the big text area.
  • 6
    Save it ("Commit") Scroll to the bottom and click the green "Commit changes" button. You don't need to change anything else.
  • 7
    Wait about 1 minute, then visit your page! 🎉 Your page will be live at:
    altitudeaico.github.io/goteamolatoyes/hackathon/YOURNAME.html
🎉 Example: If Elsie saves a file called elsie.html, her page will be at:
altitudeaico.github.io/goteamolatoyes/hackathon/elsie.html
⚠️ Something went wrong? Don't worry! Ask Uncle B or your parent to check — it's usually just a tiny typo in the filename. The most common mistake is forgetting the .html at the end.
✏️

Step 3 — Update your page any time

Want to make your page even cooler? Easy! Go back to Claude, make changes, copy the new code, then:

  • 1
    Find your file on GitHub Go to the hackathon folder and click on your .html file.
  • 2
    Click the pencil ✏️ icon This opens the editor so you can change the code.
  • 3
    Select all, paste, commit Press Ctrl+A (or Cmd+A on Mac) to select everything, then paste your new code. Click "Commit changes" to save.
💡 Pro tip: You can ask Claude to make lots of small changes one at a time. Each time Claude updates it, copy the new full code and paste it over the old one on GitHub.
💬

Useful Claude prompts

Here are some prompts you can use to make your page better:

🎨 Change colours or style:

Can you update my webpage to use [COLOUR] as the main colour, and make the background have animated stars?

➕ Add something new:

Add a section to my webpage called "My Favourite Things" with a list of my top 5 favourite [movies / foods / games]. Make it look fun with emojis.

🔗 Add a link to your team quiz (later):

Add a big button at the top of my page that says "Play Our Quiz! 🎮" and links to: [PASTE YOUR QUIZ LINK HERE]
